Sudoku in C!

This was my second undertaking, a command line based sudoku solver written in C.

I've been learning to code in C at work and the raison d'etre for the solver was to put into practice what i'd learnt - that, and I needed a Sudoku cheat code!

It served as a gentle introduction to algorithms (implementing a backtracking algorithm) and definitely helped consolidate some of the basics.

I'm not quite done with it yet, mind - i'd love to design a GUI for it soon!
