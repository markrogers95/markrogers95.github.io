The purpose of this website is to detail a little about myself and a little about the projects i'm putting together in my free time.

Dotted around the site are links to my socials, including my Github, but this sections serves to provide a brief outline of what i've managed so far and what I intend to get round to in the not so distant future!

That being said, get clicking to find out more..

This website was the first project I started work on and, as previously mentioned, one of its purposes is to provide a little background on both me and the projects I'm working on.

However, whilst the above is well and good the real reason for putting this website together was to start on a learning path towards web-development!

I've created this website as a first effort in an about as vanilla way as possible. I've got to grips with HTML, CSS and Javascript in a decent way with a little JQuery and PHP thrown in for good measure. I've tried not to rely on Bootstrap, and although it is definitely present, its usage has been minimal. I'm still getting to grips with targeted layout according to device screen size but that'll be sorted soon. The contact form has all the bells and whilstles required to process server side and send an email without the need for your own client but due to the nature of Github's hosting that is on hold for now!

It's primitive now, but it won't be for long! Watch this space..

Project number two was a command line Sudoku solver written in C. I wanted to do something that would help consoliate the laguage basics and also maybe also get a little use when I was done!

I knew I wanted to practice the basics like reading/writing from/to file, pointer/array arithmetic and linking and compiling different files. I tried to include some basic error handling and redundancy but didn't go crazy here.

I got a little taste of algorithms whilst working on this project as it implements a fairly simple backtracking algorithm that explores every solution in sequence up until failure or a successful solve.

Again this is a project I hope to revisit in the not-so-distant future, moving away from the command line and towards a more visual approach, perhaps by way of a simple web app in React!

Moving forward here's a few ideas on what I would like to have a go at next..

I'm currently making my way through T.H.Cormen and Co.'s "Introduction to Algorithms" and as such would quite like to try my hand at implementing a few of these and present their working visually with a web app! I've enjoyed the sections on Sorting Algorithms and Red-Black trees so will probably look to one or both of these as project ideas next!

Something else that I find really interesting, more as an aside, is game programming with C++. I've seen a couple of neat C++ games kicking about and would love to try my hand and getting my own up and running!

Lots to look at here and there, and certainly enough to keep me busy through lockdown and beyond!
